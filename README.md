#安装方法

1.安装Vusial Studio 2019 及以上版本，安装.NET桌面开发组件 安装 .net5 运行时组件。<br/>
2.安装Mysql并导入项目中的data.sql文件。<br/>
3.编译运行，客户端使用上面的版本。<br/>
4.本项目依赖于NetWorkLibrary。<br/>
